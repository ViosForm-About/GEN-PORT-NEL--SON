## 🔧 Troubleshooting

### Stats not updating?
- Server mode: Check `server.js` is running
- Static mode: Uses fallback demo data (no backend required)

### Images not loading?
- Skills use CDN logos (devicons)
- Fallback SVG included for offline support

### Deployment errors?
- Vercel: Ensure `vercel.json` is present
- Node version: Requires Node 18+
- Static deploy: Remove `server.js` dependency if using pure static hosting
